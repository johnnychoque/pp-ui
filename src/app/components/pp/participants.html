This tool allows you to select the experiments in which you want to participate

Selected Experiments

This option enables you to see the list of experiments in which you have already accepted to participate.
It shows all the experiments in which you participate, either accepting to participate when you have received an invitation or you have registered directly as a participant in a public experiment.

Go To Selected Experiments

Invitations received

Based on your user profile information, experimenters can select you as a potential participant in their experiments and send you an invitation to join their group of participants.
This option enables you to view the list of invitations that have been sent to you, being able to accept or reject them.

Go To Invitations

Public Experiments

A public experiment is one in which a user can participate without needing to receive an invitation from the experimenter.
This option enables you to view the list of public experiments and select which one to participate in.

Go To Public Experiments
